Determinism

One renderer, two surfaces.

render_ai_bom in crates/sigil-core/src/aibom.rs is a pure function of the parsed AiBom struct — no I/O, no clocks, no globals. The wasm crate exposes a single entry point that calls it. A native Rust test in crates/sigil-aibom-wasm/tests/determinism.rs asserts wasm-rendered output is byte-for-byte equal to the native render for every committed sample, and CI runs the committed .wasm against the same fixtures so the deployed bundle cannot drift from the source.
